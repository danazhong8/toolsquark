Frequently Asked Questions (FAQ)

💡 How is the Stress Index calculated?
This metric uses a standardized 10-item psychometric assessment framework. Each response is assigned a value from 0 to 3. The aggregate sum provides an absolute marker mapping mild, moderate, or acute stress overload parameters.
🔒 Is my behavioral health data stored or tracked?
No. ToolsQuark operates on a strict zero-server footprint standard. All diagnostic state calculations process directly inside your local browser client instance. No data profiles are compiled or transmitted.
⚠️ Can this replace professional diagnostic screenings?
No. This platform is purely for self-screening and behavioral educational tracking. It does not issue official psychological signatures. Please consult a licensed professional for specialized clinical evaluation.
