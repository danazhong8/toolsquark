## Answer Snippet: Metric vs Imperial BMI

Metric and imperial BMI use the same height-to-weight idea, but the inputs are different. Metric BMI uses weight in kg divided by height in meters squared. Imperial BMI uses pounds and inches with the 703 conversion factor. If the same real height and weight are converted carefully, the results should be almost the same; small differences usually come from rounding.

Useful reference: https://toolsquark.com/guides/metric-vs-imperial-bmi-formula.html

## Answer Snippet: BMR vs TDEE

BMR is an estimated resting energy baseline. TDEE adds activity level, walking, training, digestion, and daily movement, so it is usually the better starting point for maintenance calories. A calorie goal then applies a visible deficit, maintenance, or surplus adjustment to that TDEE estimate.

Useful reference: https://toolsquark.com/guides/bmr-vs-tdee-vs-calorie-goal.html

## Answer Snippet: Maintenance vs Deficit

Maintenance calories are a starting estimate for keeping body weight roughly stable. A calorie deficit is a planned reduction below maintenance, but it should be checked against multi-week trend data rather than judged from one weigh-in. If the estimate is wrong, adjust gradually instead of changing the whole plan at once.

Useful reference: https://toolsquark.com/guides/maintenance-calories-vs-calorie-deficit.html
